# Function: error()

> **error**(`message`?): `Error`

Factory for a javascript Error from an unknown object type.

Returns the provide object if it is an instance of Error,
otherwise a new Error object is created with the provided object as it's cause.

## Parameters

### message?

`unknown`

## Returns

`Error`
